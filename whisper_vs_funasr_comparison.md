### 常见错误

#### Whisper tiny 常见问题:
1. **同音字错误**: "彭" vs "朋"、"元方" vs "远方"
2. **词组拆分**: "本来木" (应为 "本栏目")
3. **成语/专业术语**: "小卓微薰" (应为 "小酌微醺")
4. **标点缺失**: 大部分地方没有标点
5. **格式问题**: 每句话独立一行，阅读不连贯

#### FunASR SenseVoice 问题:
1. **特殊标签**: 输出包含 `<|zh|><|NEUTRAL|><|BGM|><|withitn|>` 等标签
2. **文本长度**: 比Whisper短约50%（可能压缩了重复内容或进行了句子合并）
3. **分段问题**: 整篇文章几乎没有分段，全部连在一起

## 优缺点总结

**优点**:
- ✅ 提供精确时间戳
- ✅ 模型轻量，快速加载
- ✅ 多语言支持广泛

```python
# 修改 transcriber.py，使用 FunASR 作为主引擎
def transcribe(self, audio_path: str) -> List[Dict]:
    # 使用 FunASR SenseVoice
    from funasr import AutoModel
    
    model = AutoModel(
        model="iic/SenseVoiceSmall",
        vad_model="fsmn-vad",
        device="cpu"  # 或 "mps" for Apple Silicon
    )
    
    result = model.generate(
        input=audio_path,
        language="zh",
        use_itn=True
    )
    
    # 后处理：去除标签
    text = self._remove_tags(result)
    
    # 智能分段
    paragraphs = self._smart_split(text)
    
    return paragraphs
```
